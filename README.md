# AI Mock Interview Platform

DEPI Graduation Project

## Description

An AI-powered system that simulates real interview experiences using:

- Multi-agent architecture
- Real-time voice interaction
- CV understanding
- AI-based evaluation

## Tech Stack (Planned)

- React
- Node.js
- FastAPI
- LangGraph
- Groq
- STT / TTS
